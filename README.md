# اذا جوكم بيض 🥚

هذا ملف ZIP كامل يحتوي على المشروع بشكل منفصل:

## المجلدات
- `frontend/` الواجهة
- `backend/` الخادم وقاعدة البيانات المحلية

## التشغيل

### 1) تشغيل الخادم
```bash
cd backend
npm install
npm start
```

### 2) تشغيل الواجهة
افتح:
```bash
frontend/index.html
```

## حساب الادمن الافتراضي
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

## أنواع الحسابات
- `admin` ادمن كامل
- `supervisor` مشرف بصلاحيات يحددها الادمن
- `subscriber` مشترك مدفوع بصلاحيات أعلى من العضو
- `member` عضو عادي يمكنه اللعب مع أصدقائه بصلاحيات محدودة

## رفع الأسئلة دفعة واحدة
الادمن أو من يملك صلاحية إدارة المحتوى يستطيع رفع ملف:
- TXT
- PDF
- DOCX

### التنسيق المدعوم
```text
السؤال:
-----------
ما عاصمة فرنسا؟
1- باريس
2- لندن
3- مدريد
4- روما

السؤال:
-----------
كم عدد أيام الأسبوع؟
1- 7
```
